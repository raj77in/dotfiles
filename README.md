# dotfiles
Dotfiles
[My Blog](http://blog.amit-agarwal.co.in/)

My dotfiles. I am currently using i3-gaps on Fedora (self compiled :) )

I use rofi for menu and variety for wallpaper
You might also want to install pywal

## How to use
```
git clone https://github.com/raj77in/dotfiles
cd dotfiles
git submodule init
git submodule update

```


## Install pywal
Local install :
```pip install --user pywal```

Global install :
```pip install pywal```
